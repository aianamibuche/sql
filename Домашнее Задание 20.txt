1. Обработчик на /todos/new/<title>
Запросы на этот обработчик должны сохранять title в список todos

2. Обработчик на /todos
Возвращает список заданий

3. Обработчик на /todos/remove/<index>
Удаляет по индексу. Возвращает обновленный список. Нужно учесть, что индекса может не существовать

4. Обработчик на /todos/get/<index>
Получает по индексу. Возвращает задание по индексу. Нужно учесть, что индекса может не существовать, тогда возвращать "Задание не найдено"

5. Обработчик на /todos/edit/<index>/<new_title>
Редактирует по индексу. Заменяет задание по заданному индексу на новое название. Возвращает обновленный список
